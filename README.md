# nix-devbox

Portable Nix configuration for managing development tools across machines.

## Overview

This repository contains Nix configurations for setting up a consistent development environment across any macOS machine with access to this repository. Using Nix, all development tools and dependencies are declaratively managed and reproducible.

## Prerequisites

- macOS
- [Nix package manager](https://nixos.org/download.html) installed
- Git access to this repository
